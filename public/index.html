Privacy-first availability

Share when you’re busy—nothing else

Automatic: connect iCloud with an app-specific password and this server will re-fetch your calendars every few minutes. Manual: upload an .ics export. Either way we only publish generic “Busy” blocks—no titles, locations, or notes.

1. Connect iCloud (recommended)

Use an app-specific password from appleid.apple.com (sign in → Security). Your normal Apple ID password will not work here. The server encrypts the app password at rest; you must set CREDENTIAL_ENCRYPTION_KEY (see README).

2. Or upload an .ics file

First time: leave “Update” unchecked. To push a new export to the same subscribe URL, check “Update” and paste your manage key.

Your links
